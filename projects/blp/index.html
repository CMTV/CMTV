Blank Line Parser
Конвертация любого текста в массив блоков.
Статус
Разработал
Время
2 д 8 ч
Исходный код

Абстрактный парсер текстовых файлов, который разбивает их на блоки и инлайнеры. На блоки текст разбивается по пустой строке. Внутри блока весь текст разделяется на массив инлайнеров.

Можно добавлять собственные типы блоков и инлайнеров. Есть встроенная поддержка yml-объектов внутри текста.
